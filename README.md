
The More Balatro Mod is a content mod that adds Custom Jokers to Balatro

# Content

The mod currently adds:
- 9 Jokers
    The Stranger
    The Two Man
    The Crib
    The Sequence
    The Monarchy
    The Squad
    The Frat
    The Gang
    Yotie

# Requirements
- [Steamodded](https://github.com/Steamopollys/Steamodded)
- [Lovely](https://github.com/ethangreen-dev/lovely-injector)
